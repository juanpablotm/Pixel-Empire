# Marca — Pixel Empire: Game Studio Tycoon

Assets de logo en vector (SVG), listos para usar. Escalan sin pérdida de banner a favicon.
Son de la **Fase 7** (pulido); no bloquean el desarrollo.

## Archivos

| Archivo | Uso |
|---------|-----|
| `pixel-empire-logo-dark-bg.svg` | Lockup horizontal para **fondos oscuros** (texto claro) |
| `pixel-empire-logo-light-bg.svg` | Lockup horizontal para **fondos claros** (texto oscuro) |
| `pixel-empire-mark.svg` | Solo el emblema (corona pixelada). Favicon / icono pequeño |
| `pixel-empire-app-icon.svg` | Emblema sobre azulejo oscuro redondeado. Icono de app (512×512) |

## Concepto

La **corona pixelada** funde las dos ideas del nombre: los bloques = "pixel", la corona = "empire".
El acento **dorado** representa el eje *Capital*; el pequeño cuadrado **verde** es un guiño a
"greenlight" / integridad (el otro extremo del dilema moral).

El wordmark usa una tipográfica **monoespaciada** (guiño al código / game dev) en minúsculas, con el
descriptor `game studio tycoon` debajo.

## Notas

- Las fuentes van como *stack* del sistema (mono y sans) para que rendericen en cualquier parte. Si
  más adelante eliges una tipográfica propia, se sustituye en los `font-family`.
- Para exportar a PNG (p. ej. iconos de tienda), abre el SVG en el navegador o en un editor vectorial
  y exporta al tamaño deseado; el vector no pierde calidad.
- El nombre es texto real editable dentro del SVG: cero problemas de tipografía deformada (a diferencia
  de un logo generado por IA).
